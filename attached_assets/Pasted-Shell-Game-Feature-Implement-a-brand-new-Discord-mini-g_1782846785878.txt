Shell Game Feature
Implement a brand-new Discord mini-game called Shell Game.
What is Shell Game?
Shell Game (also known as Three Cup Monte) is a classic observation and memory game.
At the beginning of each round, a small ball is placed underneath one of several upside-down cups.
The player is shown exactly where the ball is before the game starts.
After a short delay, all cups cover their positions and begin shuffling around.
The player’s objective is to carefully follow the cup containing the ball during the shuffle.
When the shuffle stops, the player must select the cup they believe still hides the ball.
If they choose correctly, they win a reward.
If they choose incorrectly, they lose.
The game should reward observation and memory, not pure luck.

Gameplay Flow
Step 1 — Show the Ball
When the game starts:
Display all cups.
One cup is lifted so the player can clearly see the ball underneath.
Keep the ball visible for about 2 seconds.
This allows the player to know exactly where the ball starts.
 
⸻
 
Step 2 — Cover the Ball
The lifted cup closes.
Every cup is now upside down.
The shuffle immediately begins.
 
⸻
 
Step 3 — Shuffle Animation
Do NOT calculate cup movements in code.
Instead:
Randomly select one pre-rendered shuffle animation.
Each animation already knows where the ball ends.
Every animation includes metadata such as:
winningCup = 2

The game logic must only read this value.

Never attempt to calculate or analyse the animation.

The animation should be smooth, easy to follow and fair.

Players should genuinely be able to track the correct cup if they pay attention.

Different shuffle patterns should exist so the game does not become repetitive.

⸻

Step 4 — Player Choice

When the shuffle finishes:

Edit the SAME Discord message.

Display buttons allowing the player to choose a cup.

Easy

3 cups

Medium

4 cups

Hard

5 cups

Only the player who started the game may interact.

If another player presses a button:

Send an ephemeral message explaining that the game does not belong to them.

Players have 15 seconds to make a choice.

If no choice is made:

* disable every button
* automatically reveal the correct cup
* end the game

⸻

Step 5 — Reveal Animation

After the player selects a cup:

Edit the SAME Discord message.

Play a reveal GIF.

Never use static reveal images.

If the player selected the correct cup:

* lift the selected cup
* reveal the ball underneath
* display a victory message
* award a reward

If the player selected the wrong cup:

* first lift the chosen cup (empty)
* then lift the correct cup
* reveal the ball
* display a defeat message

Disable every button after the game finishes.
